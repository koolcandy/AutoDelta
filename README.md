
# AutoDelta

基于 `scrcpy` + 模板匹配的自动倒子弹工具

## 模板采集（必须）

运行模板选择器，**在游戏界面中截取按钮/文字区域**：

```bash
python template.py
```

模板将保存到：

- 图片：`templates/<模板名>.png`
- 坐标：`templates/coords.json`

> 需要保证模板名与脚本中使用的名称一致（如 `交易行`、`开始行动`、`确认配装` 等）。

## 使用方法

```bash
python bot.py
```
